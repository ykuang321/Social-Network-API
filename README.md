  ## Installation
  To install the necessary dependencies, run the following commands:
  
  ```
  npm install express
  npm i moment
  npm install mongoose
  ```
  
  ## Usage
  After install the necessary dependencies, run below command in terminal to start the application:

  ```
  node index
  ```
